== Description ==

Module de paiement Atos SIPS (majorité des banques françaises) pour le plugin WP e-Commerce
ATOS SIPS est la technologie utilisée par de nombreuses banques françaises :

Banque Populaire (CyberPlus™), Société Générale (Sogenactif™), Crédit Lyonnais (Sherlock’s™), Crédit du Nord (Webaffaires™), CCF (Elysnet™), BNP (Mercanet™)

Plugin développé par Thomas de l'agence web [6WWW](http://6www.net "Agence web à Reims, Rennes, Paris et Lyon")

Requis : Plugin WP e-Commerce téléchargeable ici : http://wordpress.org/extend/plugins/wp-e-commerce/

== Installation ==

Pour des informations détaillées rendez-vous sur http://wpcb.fr/doc

En quelques mots :

1. Vérifier que le plugin WP e-Commerce est installé, sinon télécharger le ici : http://wordpress.org/extend/plugins/wp-e-commerce/
2. Télécharger `wpcb` vers `/wp-content/plugins/` par ftp
3. Activer le plugin à travers le menu 'Extensions' dans votre tableau de bord WordPress
4. Placer le shortcode `[atos]` sur une page vierge (cette page ne doit pas avoir de page parent ex : http://monsite.fr/paiement-atos
5. Régler le plugin dans `Réglages->Boutique->Paiement->Atos`
